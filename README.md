# material-noobie
Create dead simple Material-UI without billions of node dependencies.

# Setup
```
npm install
npm update
npm run build
```

# Structure
```
├── client
│   └── index.js
├── config
│   └── webpack.js
├── package.json
└── site
    ├── css
    │   └── style.css
    ├── images
    ├── index.html
    └── js
        └── main.bundle.js
```
