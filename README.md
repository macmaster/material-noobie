# material-noobie
Create dead simple Material applications without billions of node dependencies.

# Setup
```
npm install
npm update
npm run build
```

# Structure
```
├── client
│   ├── index.js
│   ├── nav
│   │   └── AppBar.js
│   ├── store
│   │   ├── actions
│   │   │   └── items.js
│   │   ├── index.js
│   │   └── reducers
│   │       ├── index.js
│   │       └── items.js
│   └── style
│       └── theme.js
├── config
│   └── webpack.js
└── site
    ├── css
    │   └── style.css
    ├── images
    │   └── leaf.png
    └── index.html
```
